# 虚灵

### 简介

虚灵是一个稀有度等级，范围从 1000 亿到 9990 亿。目前游戏中共有 34 种不同的虚灵矿石，最稀有的虚灵是洞穴专属的 🇧🇶，基础稀有度最高的虚灵是 🇫🇯，最常见的虚灵是 ⚖️。在子领域一中，虚灵的生成消息是："一种不可理解的造物让你陷入了恍惚……"
