# 古董

古董是可在世界一、世界二、水水世界和子领域一中获得的矿石等级。它被称为游戏初期"最容易获得"的矿石等级，因为该等级的大多数矿石稀有度不超过 200 万。

目前共有 16 种古董矿石。其中 9 种在世界一和水水世界，都是爱心形状；4 种在世界二，都是书本形状；2 种在子领域一，都是旗帜；最后一种是限时矿石，曾在六月、七月和八月期间可获得——那就是排球。
