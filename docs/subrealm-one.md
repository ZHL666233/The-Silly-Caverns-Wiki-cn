# 子领域一

这是笨猫探穴的第一个子领域，通常是你进入的第二个或第三个世界。这个世界有 7 个普通地层，全部基于世界各大洲命名（南极洲除外）。这个世界只有一把镐子——生命之树，它有 5 次升级可以强化。这里有 7 个普通齿轮，都提供小幅增益。这个世界只允许使用镐子生命之树和虫洞灭绝者（仅限合成未知之球后）。

建议在获得无限收集器后再来这里，这里只需获取前两个齿轮。之后就可以暂时不管这个世界，直到你完成[世界一](./world-one.md)和[世界二](./world-two.md)。因为要获得这里的奖杯，需要将生命之树升到最高级，并找到稀有度为 1/12.5T 的矿石 🏁。
