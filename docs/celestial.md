# 天界

### 简介

天界是一个稀有度等级，范围从 1 起上不封顶。目前游戏中共有 48 种不同的天界等级矿石，最稀有的天界矿石是[全能矿洞之神](./omnipotent-god-of-the-mine.md)，而 🩶 和 🩷 是最常见的天界矿石。

获得天界矿石的方法有很多，但其中相当一部分是通过在[世界一](./world-one.md)中触发"天界地层触发器"获得的。要找到触发器，玩家需要不断向下挖掘，直到收到消息：*"在世界无尽的深处，早期环境的重复构成了下层大地，然而……一种奇怪的陌生感笼罩了你……"* 同时伴有音效。一旦触发，玩家需要在激活的地层中挖掘来获取天界矿石。
