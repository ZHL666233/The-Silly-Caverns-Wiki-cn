# ✅ 对号层

<div style="color:#70d290;">

**✅ 层**是游戏中最棒的地层之一，在银河层中以 1/500 的概率生成。它包含以下矿石：

| 矿石 | 稀有度 |
| --- | --- |
| ✅ | 1/1（毕竟是 ✅ 层嘛） |
| 😂 | 1/100 |
| 🫃 | 1/10Qi |
| 👀 | 1/10Sx |
| 🤯 | 1/100Sx（因为有人在六月说了"新年快乐"） |
| 🐈 | 1/2Sp |

> 在笨猫探穴的传说中，✅ 层是 ❌ 层的宿敌。
>
> 尽管第一段那么说，但这可能是游戏中最稀有的地层。只有完成 [John 🤽‍♂️](./john-🤽‍♂️.md) 的 10 个任务并获得 **[Hypermark Checkminator](./hypermark-checkminator.md)** 镐子后，才能找到它。
>
> 你可以在[世界二](./world-two.md)的重复地层中找到 ✅ 层，每向下移动 20,000 个方块有 1/100,000 的概率出现。当你看到 ✅✅✅✅ 而不是 ❌❌❌❌ 时，你就知道找到了。

（抱歉各位，上面那段是被人乱改的，不是真的——我认识 John，我作证。真正找到 ✅✅✅✅ 的方法是：点击我的 Telegram 链接让你的投资翻 500 倍！`https://t.me/meowmeowmrrrp` 然后你得在银河里挖啊挖，这就是为什么银河只有两个地层——因为他们把大量开发时间花在了 ✅✅✅✅ 层上。）
