| 属性 | 值 |
| --- | --- |
| 镐子名称 | <span style="color:#a9a9a9;font-weight:bold">腐土槌</span> |
| 镐子等级 | <span style="color:#f8480f;font-weight:bold">等级 1</span> |
| 属性 | 技能挖掘 48 / 技能揭示 2.8. / 1/50 技能触发率 / 一致性 2.8. |

### 简介

腐土槌是可在[世界一](./world-one.md)中找到的镐子。其技能可挖出一个十字形，并在技能每侧额外挖掘 3 个方块。该技能可挖掘 48 个方块，同时揭示 140 个方块，拥有 1 倍幸运，挖掘方块时有 1/50 概率触发技能。综合来看，该镐子的一致性为 2.8。
